# Factorial
```
// 표기: N!
3! = 3 * 2 * 1 = 6
```

| N | N! |
| ---- |:------------:|
|  0   |          1  |
|  1   |          1  |
|  2   |          2  |
|  3   |          6  |
|  4   |         24  |
|  5   |        120  |

### 심화(경우의 수)
N개의 수에서 K개를 선택할 수 있는 경우의 수는?
```
// 답안 공식
N! / (K! * (N - K)!)
```

위의 예시와 같이 팩토리얼은 경우의 수를 구할 때 매우 유용하다.
